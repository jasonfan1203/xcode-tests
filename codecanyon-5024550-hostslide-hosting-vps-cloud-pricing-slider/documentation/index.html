Installation

HostSlide is powered by jQuery and jQuery UI. Most templates you see nowadays include jQuery by default. HostSlide requires jQuery1.6+.

File Structure

You should upload all files listed below (from the 'code' folder) to your website:

  • css/jquery-ui.structure.min.css
  • css/jquery-ui.theme.min.css
  • js/jquery-1.11.2.min.js
  • js/jquery-ui.min.js
  • js/hostslide.min.js
<head>
    <link href="css/jquery-ui.structure.min.css" rel="stylesheet">
    <link href="css/jquery-ui.theme.min.css" rel="stylesheet">
</head>
<body>
    <script src="js/jquery-1.11.2.min.js"></script>
    <script src="js/jquery-ui.min.js"></script>
    <script src="js/hostslide.min.js"></script>
</body>

Creating a slider

Creating Plans

Adding plans is as simple as creating an array and add subarrays under it, like so:

The subarrays has to start with 0 and increase sequentially. Each subarray represents one type of resource (eg. disk space or ram) instead of a plan.

var sliderplans = [];
sliderplans[0] = ["2", "3", "4", "5", "6", "7"];
sliderplans[1] = ["10", "25", "35", "45", "60", "75"];
sliderplans[2] = ["300", "500", "700", "900", "1350", "1500"];
sliderplans[3] = ["$5", "$10", "$20", "$30", "$50", "$70"];

Creating The Slider

HostSlide is a jQuery plugin and therefore it is easy to create a slider:

Function: 
.hostslide(type, plans, currentplan, id, orientation)
Variable Description
type (string) The type of the slider. Valid values are single and multiple.
plans (array) The variable which contains the plans array as discussed above.
currentplan (number) The default plan (eg. 1, 2, 3, etc)
id (string) The class prefix of slider elements.
orientation (string) Slider orientation. Valid values are horizontal and vertical.
Example: 
<script>
$(".hs").hostslide('single', sliderplans, 3, 'hs', 'horizontal');
</script>
We will now add the HTML elements which will display the slider and the plans. In this demo, the class prefix we used was hs so we will use it in our class names like so:
Single Slider: 
<div class="hs"></div> <!-- Slider -->
<span class="hs0"></span> <!-- Plan subarray 0 -->
<input type="hidden" name="hs0" class="hs0"> <!-- A hidden input for submitting forms -->
<span class="hs1"></span> <!-- And so on... -->
<input type="hidden" name="hs1" class="hs1">
<span class="hs2"></span>
<input type="hidden" name="hs2" class="hs2">
<span class="hs3"></span>
<input type="hidden" name="hs3" class="hs3">
Multiple Sliders: 
<div class="hs0"></div> <!-- Slider -->
<span class="hs0data"></span> <!-- Plan subarray 0 -->
<input type="hidden" name="hs0data" class="hs0data"> <!-- A hidden input for submitting forms -->
<div class="hs1"></div> <!-- And so on... -->
<span class="hs1data"></span>
<input type="hidden" name="hs1data" class="hs1data">
<div class="hs2"></div><
<span class="hs2data"></span>
<input type="hidden" name="hs2data" class="hs2data">
You can now add custom stylings and texts to match your website. HostSlide detects and adds values to input tags, and html to other tags like div, span and p.
If you are using the vertical slider, make sure you add float: left and appropriate paddings in your stylesheet.

Style Customizations


You can edit styles simply by adding custom css for .ui-slider-handle, .ui-slider-range and .ui-slider

You can also use jQuery UI's ThemeRoller to make custom themes.



Order Link Redirection
